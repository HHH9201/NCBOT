# SpeakRank 插件使用说明

## 快速开始

1. **基本命令**
   - 在群里输入 `今日发言榜` 查看今日排行榜
   - 输入 `昨日发言榜` 查看昨日排行榜  
   - 输入 `总发言榜` 查看总排行榜
   - 输入 `我的发言` 查看个人统计

2. **个人统计显示**
   ```
   📊 用户12345 的发言统计
   总发言（156）
   昨日发言（23）
   今日发言（8）
   ```

3. **排行榜显示**
   ```
   🏆 总发言排行 🏆
   🥇1. 用户123: 256次
   🥈2. 用户456: 189次  
   🥉3. 用户789: 167次
   4. 用户111: 145次
   5. 用户222: 123次
   ```

## 每日自动功能

- 每天0点会自动发送昨日发言排行榜到所有有数据的群组
- 需要BotAPI支持自动发送功能

## 数据管理

- 数据会自动保存到SQLite数据库
- 可以手动输入 `保存发言数据` 立即保存
- 数据库文件位置：`/home/hjh/BOT/NCBOT/mydb/mydb.db`

## 注意事项

- 统计从插件安装开始计算
- 每日0点会重置当日统计
- 昨日数据会保留，可查询历史记录
